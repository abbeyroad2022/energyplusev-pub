생체 인증 도움말

생체인증 사용이 안되시나요?
  1. 생체 인증이 가능한 기기만 사용 가능하며, 별도의 해킹을 한 경우 사용이 불가합니다.
  2. 새로운 기기에서 생체인증 설정 시 기존 기기의 설정은 해제됩니다.
    안드로이드 6.0 이상, IOS 아이폰 5S 이상
  3. 소지한 디바이스에 생체인증이 등록되어야 사용 가능합니다.

GS&Point 자동 결제 도움말

GS&Point 자동 결제란 무엇인가요?
  1. 적립하신 GS&Point를 충전결제시 차감하는 결제하는 방식입니다.
  2. 충전 후 결제시 보유하신 GS&Point 차감 후 잔여금액은 등록한 결제정보로 자동 결제 됩니다.
